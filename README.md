# Otogryzonie Backend

## General info
A website which gives users the opportunity to adopt or buy a rodent. The project is still in development.

## Technologies
* Node.js
* Typescript
* Nest.js
* PostgreSQL
* prisma

## Setup
* Install all neccessary dependencies:<br/>

        npm install

* Run the server locally:<br/>
   
        npm run start:dev 
  
* Run tests:<br/>
   
        npm run test
